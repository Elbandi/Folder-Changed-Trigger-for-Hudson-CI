Regular expression for excluding files to being monitored. The files matching this pattern will not affect build triggering. (Leave blank to monitor all files)
Examples:

^.+\.((tmp)|(temp))$ : For excluding files ending with .tmp or .temp
^\.(.+)$ : For excluding files starts with a .
((\.(.+))|(.+\.((tmp)|(temp)))) : A combination for excluding files which start with a . or ends with .tmp or .temp
